Yolele website review

Design: 7/10

The website makes effective use of vibrant colors and a hand-drawn typography to give the products a flavorful and refreshing appeal. I think the website can improve the recipes page by displaying pictures of the recipes rather than their names since the names are rather long to read.

Usability: 6/10

The website is minimal which makes it easy to know what options are available for the user to click on. The menu displays the titles of the pages very clearly so the user does not get lost within the structure of the website. However, some interactive elements in the website do not work properly or have a delay before they start moving along with the cursor. Also, when I change the size of my window, the images which remain the same size obstruct the text which shrink in size behind them. Therefore, some the interactive elements impede the user experience.

Creativity and Content: 7.5/10

Overall, I think that the website is creative because it uses interactive elements to engage the users (although these interactive elements can still be refined). It not only features the products, it also creates added value for the users who can explore how the products can be used in recipes and who can read about the product in the blog section.
